# intro-to-express-server
A simple node web application that calculate things (using express and body-parser)

## Add calculator：
Visit http://localhost:3000 and type in two numbers and submit the form, you should be able to see the sum. 

## Add BMI calculator：
Visit http://localhost:3000/bmicalculator and type in two numbers and submit the form, you should be able to see the BMI.
